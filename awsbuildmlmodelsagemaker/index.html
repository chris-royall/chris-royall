Build ML Model with SageMaker Canvas

In this project, I am building and experimenting with my first ML model using AWS SageMaker. By using SageMaker Canvas, this provide a low-code experience for the user, and is suitable for individuals who do not have coding experience. My ML model would be used to predict house prices.

First, I changed my Region from eu-west-2 to eu-west-1, as I was aware SageMaker Canvas is only available in specific Regions. I then proceeded to create a SageMaker Domain.

I then proceeded to upload data to the S3 Bucket that was created during the creation of the SageMaker Domain, and import this data to the Canvas.

I can now begin to build my ML model, by first selecting the dataset, then selecting the target column. In this scenario, since I am analysing house prices, I select the 'median_house_value' column. I then progress with a Standard Build instead of Quick Build, as I preferred more accuracy at a result of a higher build time.

Once the ML model has completed the build, I can now analyse the dataset. I am able to see 'latitude', 'longitude' and 'total_rooms' have a high impact on a house price. I will now be able to use my ML model to predict a house price by entering batch or singular data. For my testing, I have entered some random data to understand the impact.

Next time
I would add a CORS Policy to the S3 Bucket which would allow data to be synced from SageMaker Canvas to the S3 Bucket. This would allow me to upload data directly to model in SageMaker Canvas, without the requiring access to S3. Therefore, following the best practise of least privilege.

Steps

  1. First, I navigated to 'SageMaker' and selected 'Create a SageMaker domain'.

  2. I entered a 'Name' for the new Domain, a 'Name' for the User profile.
    I created a new Execution Role with access to all S3 Buckets.


  1. I navigated to 'S3' and selected the 'sagemaker-studio' Bucket.

  2. I imported the dataset and selected 'Upload'.

  3. I then navigated back to the SageMaker domain previously created, and selected 'Launch', then 'Canvas'.

  4. Next, I selected 'Import'.

  5. To locate the file, I selected 'S3', then selected the file, then 'Import data'.


  1. To build my model, I navigated to 'New model', added a 'Model name', then selected 'Create'.

  2. I selected the dataset I previously uploaded, then navigated to 'Select dataset'.

  3. I used the drop-down menu to select my required 'Target column'.
    In this scenario, I wanted to analyse house prices.
    I then selected 'Standard build' to get the most accurate results.


  1. I selected 'Predict'.

  2. I proceeded to run a 'Single prediction'.
    I increased the 'total_rooms' value by 25%.
